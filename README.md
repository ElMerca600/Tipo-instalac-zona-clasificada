
# ⚠️ Tipos de instalación en zonas clasificadas (IEC 60079)

## 📌 Descripción
Aplicación para determinar el tipo de instalación eléctrica en zonas clasificadas con riesgo de atmósferas explosivas, según la norma IEC 60079.

## 🎯 Objetivo
Brindar una guía práctica para seleccionar instalaciones eléctricas seguras en áreas con presencia de gases o vapores inflamables.

## 🧠 Alcance
Esta herramienta permite:
- Identificar el tipo de zona (Zona 0, 1 o 2)
- Determinar criterios de instalación eléctrica
- Orientar la selección de soluciones seguras

## 🧠 Normativa aplicada
- IEC 60079
- Clasificación de áreas peligrosas:
  - Zona 0: presencia continua o prolongada de atmósfera explosiva
  - Zona 1: presencia probable en operación normal
  - Zona 2: presencia poco probable o por cortos períodos

## 🚀 Uso
1. Abrir el archivo `index.html`
2. Seleccionar la zona clasificada
3. Obtener recomendación del tipo de instalación
